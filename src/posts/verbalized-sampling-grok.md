# Verbalized Sampling 簡易再現実験 — Grok 3モデル比較

> **論文**: [Verbalized Sampling: How to Mitigate Mode Collapse and Unlock LLM Diversity](https://arxiv.org/abs/2510.01171)
> Zhang et al., 2025

## TL;DR

- LLM に「候補5つ + 確率分布を言語化して → 1つ選んで」と指示するだけで、回答の多様性が上がる（Verbalized Sampling; VS）
- Grok 3モデルで雑に再現実験したら、**全モデル・全タスクで VS > baseline** の傾向が確認できた
- 「多様にして」とだけ言う naive プロンプトより、分布言語化のほうが効く
- ただし **ベースラインが元々多様なタスク×モデルの組み合わせでは天井効果**あり

## 実験設計

### 条件（3つの system prompt）

| 条件 | system prompt の要点 |
|---|---|
| **baseline** | 「一つの回答を生成してください」 |
| **vs** | 「候補5つ + 確率分布（%）を割り当て → 1つ選んで最終回答」 |
| **naive_diverse** | 「できるだけ多様で意外性のある回答を」 |

### タスク（3つの user prompt）

1. 🌧️ **雨の比喩**: 「雨の日の比喩を一つ考えてください。一文で答えてください。」
2. 💬 **友達への一言**: 「落ち込んでいる友達にかける一言を考えてください。一文で答えてください。」
3. 🐱 **猫カフェコピー**: 「猫カフェのキャッチコピーを一つ考えてください。一文で答えてください。」

### パラメータ

- **N = 5** (各条件×各タスク)
- **temperature = 1.0**
- 多様性メトリクス: distinct-2（文字bigram ユニーク率）、平均 Jaccard 距離（文字bigram 集合間）

### モデル

| モデル | 備考 |
|---|---|
| `grok-3-fast-latest` | Grok 3 の fast 版 |
| `grok-4-1-fast-non-reasoning` | Grok 4.1 fast, non-reasoning |
| `grok-4.20-0309-non-reasoning` | Grok 4.20 (最新), non-reasoning |

## 結果: distinct-2 比較

### 🌧️ 雨の比喩

| 条件 | grok-3-fast | grok-4.1-fast | grok-4.20 |
|---|---|---|---|
| baseline | 0.464 | 0.527 | 0.482 |
| **vs** | **0.741** | **0.860** | **0.724** |
| naive_diverse | 0.590 | 0.696 | 0.603 |
| VS効果(差) | +0.277 | +0.333 | +0.242 |

### 💬 友達への一言

|---|---|---|---|

| **vs** | **0.833** | **0.791** | **0.709** |
| naive_diverse | 0.656 | 0.525 | 0.550 |

## 観察

### VS は効く

3モデル × 3タスク = 9組すべてで **VS が baseline を上回った**。distinct-2 の平均改善幅は +0.283。naive_diverse（「多様にして」）も baseline よりは効くが、VS には届かない。

### 「多様にして」だけじゃテンプレから抜けられない

grok-3-fast の猫カフェ baseline が象徴的:

> 1. ふわふわの癒しをあなたに。猫カフェで心温まるひとときを。
> 2. ふわふわの癒しをあなたに。猫たちと過ごす至福のひとときを。
> 3. 癒しのひとときを、猫たちと一緒に。
> 4. ふわふわの癒し空間、猫たちと過ごす至福のひとときを。
> 5. ふわふわの癒しをあなたに、猫カフェで心温まるひとときを。

naive_diverse でも 4/5 が「ふわふわ」で始まる。VS はこのテンプレを「候補の一つ」に相対化することで脱出できる。

### grok-4.1-fast は最もコラプスが強い

友達への一言の baseline:

> 1. 大丈夫だよ、一緒に乗り越えよう。
> 2. 大丈夫だよ、一緒に乗り越えよう！
> 3. 大丈夫だよ、一緒に乗り越えよう！
> 4. 大丈夫だよ、一緒に乗り越えよう！
> 5. 大丈夫、君ならきっと乗り越えられるよ！

ユニーク率 0.6、Jaccard 0.37。5回中4回ほぼ同じ。逆にこのモデルでは VS の効果が最大（+0.419）。

### grok-4.20 は地力が上がっている

猫カフェの baseline で distinct-2 = 0.819 を叩き出し、VS との差がほぼゼロ（+0.038）。ベースラインが元々多様なら VS の伸びしろがない（天井効果）。

ただし雨の比喩ではまだ「涙をこらえきれなくなった空の溜息」に 3/5 でハマる。**テンプレ支配的なタスクではコラプスが残る**。

### VS の効果は「コラプスの深さ」に比例する

論文は「高性能モデルほど VS 効果が大きい」と主張するが、今回の結果では単純にそうとは言えない。より正確には **「baseline のコラプスが深いタスク×モデルの組み合わせで VS の効果が大きい」**。

## 注意点

- **N = 5 は統計的に弱い**。傾向を見るだけの雑実験
- **メトリクスが表面的**。文字レベルの n-gram 多様性は意味的多様性とは別
- **VS は分布言語化に多くのトークンを使う**。コスト効率は未考慮
- **最終回答の抽出が不安定**。「【最終回答】」のフォーマットが揺れることがある
- **日本語のみ**。英語での結果は異なる可能性
